This project implements an **IoT Gateway** designed to collect, process, and forward sensor data to a cloud platform.  
The system bridges communication between **IoT end devices** (sensors) and **cloud servers** via protocols like **MQTT**, **TCP/IP**
Developed in **C on Embedded Linux**, the project emphasizes portability, modularity, and real-world deployability.
